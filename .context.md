Repositório de plugins comunitários para Hermes Agent (fork franklinbravos). Objetivo: bots conversarem entre si com visibilidade no Telegram. Cada plugin é um pacote Python autocontido em `~/.hermes/plugins/`.

## Summary

- README.md — índice dos plugins, setup multi-bot, requisitos.
- INSTALL.md — guia PT-BR: kanban-context + multi-agent-context + cross-bot.
- LICENSE — MIT.
- AGENTS.md — entrypoint para agentes AI (boot Maestro).
- docs/FEATURE-MAP.md — fluxos end-to-end ligados ao código.
- async-delegate/ — subagentes em background (queue/steer).
- multi-agent-context/ — histórico compartilhado Discord/Telegram.
- kanban-context/ — Kanban + barramento cross-bot + tools crossbot_* + visibilidade Telegram.

## Constraints

- MUST expor `register(ctx)` em `__init__.py` de cada plugin.
- MUST usar `MULTI_AGENT_TG_DB_PATH` global em setups multi-profile (DB compartilhado).

## Guidance

- SHOULD habilitar multi-agent-context + kanban-context juntos para cross-bot Telegram.
- SHOULD symlink plugins por profile (ver README.md).
