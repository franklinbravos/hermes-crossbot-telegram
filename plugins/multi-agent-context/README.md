# multi-agent-context 🤝

Histórico compartilhado Discord/Telegram para agentes multi-bot.

**Versão:** 2.0.0

## Instalação

```bash
./scripts/install.sh cross-bot    # inclui este plugin + kanban-context
```

## Configuração crítica

```bash
MULTI_AGENT_TG_DB_PATH=~/.hermes/data/multi_agent_tg_shared.db   # MESMO path em todos os profiles
```
