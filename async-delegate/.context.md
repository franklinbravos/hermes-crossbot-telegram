Plugin Hermes que dispara subagentes `hermes chat` em background via ferramentas, coordena estado em arquivos em `~/.hermes/async-tasks/` e injeta notificações na sessão de origem quando a tarefa termina.

## Summary

- __init__.py — implementação: tools `delegate_async`/`check_async_tasks`, watcher daemon, hooks e injeção queue/steer.
- plugin.yaml — manifesto (v1.1.0, hooks: pre_gateway_dispatch, pre_llm_call, on_session_end).
- README.md — documentação de arquitetura e uso do plugin.

## Constraints

- MUST usar apenas stdlib Python (sem dependências extras).
- MUST persistir metadados de roteamento em JSON (`_routing`) para sobreviver a restart do gateway.

## Guidance

- SHOULD usar `inject_mode: "queue"` para tarefas fire-and-forget; `steer` quando o resultado pode alterar o turno atual.
